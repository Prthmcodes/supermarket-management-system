# Supermarket Management System (JavaFX)

A GUI-based Supermarket Management System developed using **JavaFX**.

## Features
- JavaFX GUI using FXML
- Add, update, and remove products
- Display product list
- Billing and total calculation
- Object-Oriented Design
- Uses Java Collections & Generics

## Technologies Used
- Java
- JavaFX
- FXML
- CSS
- OOP Concepts

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Configure JavaFX SDK
4. Run `Main.java`

## JavaFX Setup
This project requires JavaFX SDK.
Make sure JavaFX is configured in your IDE before running.
